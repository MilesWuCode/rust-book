# 控制流程

if 表達式

```rust
fn main() {
    let number = 6;

    // 必須是bool
    if number % 4 == 0 {
        println!("數字可以被 4 整除");
    } else if number % 3 == 0 {
        println!("數字可以被 3 整除");
    } else if number % 2 == 0 {
        println!("數字可以被 2 整除");
    } else {
        println!("數字無法被 4、3、2 整除");
    }
}
```

```rust
// 在 let 陳述式中使用 if
fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };

    println!("數字結果為：{number}");
}
```

使用迴圈重複執行

Rust 提供三種迴圈：loop、while 和 for。

```rust
fn main() {
    loop {
        println!("再一次！");
    }
}
```

從迴圈回傳數值

```rust
fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2; // 回傳到result
        }
    };

    println!("結果為：{result}");
}
```

用迴圈標籤辨別多重迴圈

```rust
fn main() {
    let mut count = 0;

    // (')標籤
    'counting_up: loop {
        println!("count = {count}");
        let mut remaining = 10;

        loop {
            println!("remaining = {remaining}");
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up; // 返回到(')標籤
            }
            remaining -= 1;
        }

        count += 1;
    }
    println!("End count = {count}");
}
```

使用 while 做條件迴圈

```rust
fn main() {
    let mut number = 3;

    while number != 0 {
        println!("{number}!");

        number -= 1;
    }

    println!("升空！！！");
}
```

使用 for 遍歷集合

```rust
fn main() {
    let a = [10, 20, 30, 40, 50];
    let mut index = 0;

    while index < 5 {
        println!("數值為：{}", a[index]);

        index += 1;
    }
}

fn main() {
    let a = [10, 20, 30, 40, 50];

    for element in a {
        println!("數值為：{element}");
    }
}

fn main() {
    for number in (1..4).rev() {
        println!("{number}!");
    }
    println!("升空！！！");
}
```
